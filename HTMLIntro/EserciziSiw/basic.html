This is a basic html page. Content goes here. Why this text is displayed on the same line? Also, why all the spaces are not displayed? HTML is about meaning, non presentation. The browser will remove extra spaces and extra lines. If you want a line break, you should explicitly state that using <br>


which will create a line break.
Spaces     can be introduced with     "non breaking spaces" entities, or better in CSS.
Using <br> is not great, we should organize the content using for instance paragraph, along with container elements like <div> and <span>
