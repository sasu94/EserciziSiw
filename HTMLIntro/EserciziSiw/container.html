Container Elements

Block level: P and Div

Paragraph are block elements and as such can contain block and inline elements. Usually it contains text and other elements.

Div elements are also block elements but with a different purpose, namely to create a division of the page usually for style reasons. It is literally a container for flow content, a collection of (hopefully) more semantically marked-up content that may need to be grouped together.

Visually, paragraphs introduce a new line and additional spacing line, whereas div element do not introduce any further spacing above or below.
